Become a JavaScript expert with this in-depth guide to the new, essential tools and techniques every developer should know!

JavaScript has changed completely since it's early days of existence on the web. Join developer and keynote speaker Christian Heilmann to update your skills and unlock JavaScript's potential. From how to think about JavaScript in the coding landscape today to joining the JavaScript community, this class covers every detail for developers looking to stay on the cutting edge. You'll learn how to: Embrace the modern JavaScript development stack Set up your dev environment to prevent mistakes Learn to recognize the best JavaScript resources on the web Find the JavaScript community to get more involved Whether you've just started dabbling with JavaScript or have been working with it for years, after taking this class, you'll be coding cleaner and faster, creating better work and making products on a whole new level.
